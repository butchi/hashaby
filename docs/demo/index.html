Hashaby

#.sec-about-heading

Hashabyとは？

#.about-description

Hashabyは URLハッシュ (フラグメント識別子) を
もっと便利に変身させるライブラリです。

#.sec-about-hash

ハッシュの本来の使われ方

URLの“#”以降の文字列をキーに、 id属性かname属性に指定された要素にジャンプします。

#.sec-problem

問題点

ページ内の好きなところにジャンプできない！

#.sec-solution

そこで

魔法のライブラリ“Hashaby”を作りました！

#.sec-install

インストール方法

bower install hashaby

#.sec-function-page-jump

主な機能: ジャンプ機能

3種類の方法でページ内ジャンプができます。

#.sec-command-find-class

“.”（ドット）コマンド

クラス名を指定してジャンプ

#.sec-command-query

“?”（クエスチョン）コマンド

querySelectorでジャンプ

#?section[class="sec-command-jump"]

“=”（イコール）コマンド

式を評価してジャンプ

#="."+["sec", "calculation"].join("-")

ん？

今、計算した？

#.sec-eval-function

関数の実行

そう、関数も叩けます。

#.sec-command-exec

“;”（セミコロン）コマンド

式を実行

#;alert(5)

“+”（プラス）コマンド

設定したオブジェクト内のメソッドを実行

#+showMessage

#+showArg(5)

#+showArg("hoge")

#+showArgs(3, "piyo")

おさらい

“.”: クラス名でジャンプ

“?”: セレクターでジャンプ

“=”: 評価してジャンプ

“;”: 式を実行

“+”: メソッドを実行

#.sec-security

セキュリティー的に大丈夫なの？

“;”コマンドと“=”コマンドは設定で許可したドメインでのみ実行可

→ テストサイトで有効にして本番サーバーでは動かさないようにできる

“+”コマンドの引数は文字列と数値のみ

#.sec-function-appendix

どうしても残っちゃう"#"を除去
